توثيق النهايات

المصادقة (Authentication)

الطريقة الأولى: JWT Token

أرسل التوكن المستلم عند تسجيل الدخول في ترويسة Authorization:

Authorization: Bearer YOUR_JWT_TOKEN

الطريقة الثانية: API Key

المسار المفضل للمطورين والخدمات الخلفية:

X-Api-Key: جاري التحميل...

الجلسات (Sessions)

POST /api/sessions/create

إنشاء جلسة ربط جديدة لمزود "shamcash".

{ "success": true, "session": { "id": "uuid", "qrPayload": "qr_string", "expiresAt": "date" } }
GET /api/sessions/:id/status

التحقق من حالة الجلسة (pending, linked, expired).

المحافظ (Wallets)

GET /api/wallets

قائمة بجميع المحافظ المرتبطة بحسابك.

GET /api/wallets/:id/balance

تحديث وجلب الرصيد الحالي للمحفظة.

GET /api/wallets/:id/transactions

سجل بآخر المعاملات المالية.
